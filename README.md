# 🛡️ Cybersecurity Port Scanner

A simple Python-based port scanner that checks for open ports on a target system.

## 🚀 Features
- Scans ports 1–1024
- Supports domain and IP input
- Displays open ports

## 🛠️ Technologies Used
- Python
- Socket Programming

## ▶️ How to Run
```bash
python port_scanner.py
