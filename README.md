# 🛡️ Advanced Cybersecurity Port Scanner

A multithreaded Python-based port scanner that detects open ports and performs basic banner grabbing to identify running services.

## 🚀 Features
- ⚡ Multithreaded scanning (fast performance)
- 🔍 Scans ports 1–1024
- 🌐 Supports domain names and IP addresses
- 🧠 Banner grabbing for service detection

## 🛠️ Technologies Used
- Python
- Socket Programming
- ThreadPoolExecutor (Multithreading)

## ▶️ How to Run
```bash
python port_scanner.py
